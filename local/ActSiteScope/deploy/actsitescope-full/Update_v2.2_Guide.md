# ActSiteScope v2.2 업데이트 가이드 (v2.1 대상)

본 문서는 기존에 구성된 **ActSiteScope v2.1** 환경을 최신 3D 뷰어 안정화 버전인 **v2.2**로 업데이트하는 절차를 안내합니다. 10GB 전체 패키지를 다시 다운로드할 필요 없이 본 패치 파일만으로 업데이트가 가능합니다.

## 1. 전제 조건
- 기존 `actsitescope-full-v2.1.tar.gz`의 압축을 해제하여 설치된 디렉터리 경로를 확보하십시오.
- Docker 및 Docker Compose가 설치되어 있어야 합니다.

## 2. 업데이트 절차

### ① 기존 서비스 중지
기존에 실행 중인 컨테이너를 중지합니다.
```bash
docker-compose down
```

### ② 소스 코드 및 설정 파일 교체
본 패치 패키지에 포함된 파일들을 기존 설치 경로에 덮어씁니다.
- `app/streamlit_app.py` -> (설치경로)/app/ 에 덮어쓰기
- `docker-compose.yml` -> (설치경로)/ 에 덮어쓰기

### ③ 이미지 재빌드 (증분 빌드)
Docker의 레이어 캐싱 기능을 활용하여 변경된 소스 코드만 반영하여 빌드합니다. 이 과정은 매우 빠르게 완료됩니다.
```bash
docker-compose build
```

### ④ 서비스 재시작
새로운 v2.2 버전으로 서비스를 시작합니다.
```bash
docker-compose up -d
```

## 3. 변경 사항 (Release Notes)
- **3D 뷰어 안정화**: 대형 단백질 구조 로딩 시의 렌더링 지연 및 문법 오류 해결
- **시각화 기능 강화**: 
    - M-CSA 잔기에 대한 레이블(이름/번호) 상시 표시
    - EP(Electrostatic Potential) 수치 기반 레이블 자동 컬러링 (Blue: +, Red: -)
- **뷰어 로직 최적화**: 도킹 결과 확인 시 선택된 베이스에 따른 타겟 잔기 분리 노출

## 4. 문제 해결
업데이트 중 문제가 발생하거나 뷰어가 정상적으로 나타나지 않을 경우, 브라우저의 캐시를 삭제(Ctrl+F5)한 후 다시 시도해 주십시오.
